# 个人品牌资产盘点

> 全面评估个人品牌价值的工具，六维资产分析与增值建议

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 六维资产录入 | 受众、内容、技能、人脉、变现五大维度全面评估 |
| 受众资产 | 公众号/抖音/小红书/B站/知乎/私域粉丝数录入 |
| 内容资产 | 文章/视频/课程/爆款作品数量统计 |
| 技能资产 | 核心技能/专业度/创作能力/变现能力评分 |
| 人脉资产 | 行业大佬/同行/客户/社群影响力量化 |
| 变现资产 | 月收入/付费用户/产品数/客单价分析 |
| 综合评分 | 自动计算各维度得分（0-100分） |
| 雷达图 | 五维能力可视化对比展示 |
| 粉丝分布图 | 各平台粉丝占比饼图 |
| 短板分析 | 自动识别最弱维度并给出建议 |
| 增值建议 | 针对每个不足维度的具体改进方案 |
| 估值等级 | 从初学者到领军者7个等级评定 |

## Installation / 安装

无需安装，直接在浏览器中打开 `index.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/personal-brand-assessment.git

# 进入目录
cd personal-brand-assessment

# 在浏览器中打开
open index.html  # macOS
```

## Usage / 使用方法

### 基础用法

1. 打开 `index.html` 文件
2. 在「资产录入」Tab 填写各项资产数据
3. 点击「生成分析报告」查看评估结果
4. 关注短板分析和增值建议，制定改进计划

### 估值等级说明

| 等级 | 分数范围 | 说明 |
|------|:--------:|------|
| 初学者 | 0-14 | 刚开始建立个人品牌 |
| 入门者 | 15-29 | 已有一定基础积累 |
| 成长期 | 30-44 | 品牌资产稳步增长 |
| 成熟期 | 45-59 | 多维度均衡发展 |
| 资深者 | 60-74 | 具备较强品牌影响力 |
| 专家级 | 75-89 | 行业内有知名度 |
| 领军者 | 90-100 | 行业头部影响力 |

### 数据存储

所有数据自动保存到浏览器 localStorage，无需担心数据丢失。

```javascript
// 资产数据结构示例
{
  fans_gzh: 5000,      // 公众号粉丝
  fans_douyin: 10000,  // 抖音粉丝
  content_article: 50, // 原创 文章数
  skill_core: 8,       // 核心技能评分（1-10）
  income_monthly: 3000 // 月均收入
}
```
